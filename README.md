# FitPulse

🚀 FitPulse – AI Inspired Fitness Platform

FitPulse is a modern fitness SaaS landing page built using HTML, CSS, and JavaScript.

It features a clean modern UI, animated counters, scroll animations, and a fully responsive layout designed to simulate a real startup fitness product.
